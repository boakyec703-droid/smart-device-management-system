# smart-device-management-system
#Introduction
The Smart Device Management System is a Python Project that demonstrates the principles of Object-Oriented Programming(OOP).
The system manages different smart devices used in a smart home. Although every device has infomation such as device ID, device name, and power status, each devicehas its own features.
The project applies all four main OOP concepts:
-Encapsulation
-Abstraction
-Inheritance
-Polymorphism

#Project Objectives
The project allows users to :
-Turn devices on and off
-Adjust the brightness of a smart light
-Lock and unlock a smart door lock
-Set the temperature of a smart thermostat
